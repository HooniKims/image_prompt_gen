Chrome Extension

Auto-K 이미지 프롬프트 도우미

ChatGPT와 Gemini 입력창에 이미지 생성 프롬프트를 안전하게 넣어주는 도구

설치 방법

  1. ZIP 파일을 압축 해제합니다.
  2. Chrome 주소창에 chrome://extensions를 입력합니다.
  3. 오른쪽 위 개발자 모드를 켭니다.
  4. 압축해제된 확장 프로그램 로드를 누르고 이 폴더를 선택합니다.
  5. ChatGPT 또는 Gemini 페이지를 열면 오른쪽 패널이 나타납니다.

기본 동작

  • 프롬프트 만들기는 별도 서버 설정 없이 브라우저 안에서 바로 동작합니다.
  • 기본값은 자동 전송이 아니라 입력창 삽입입니다.
  • 기존 입력창 내용이 있으면 덮어쓰기, 이어붙이기, 취소를 선택합니다.
  • 작업 기록은 Chrome 로컬 저장소에만 저장됩니다.
  • 사용자가 고급 설정에서 직접 켠 경우에만 사용자 지정 Prompt API를 호출합니다.

고급 설정

일반 사용자는 수정하지 않아도 됩니다. 직접 관리하는 Prompt API가 있을 때만 사용합니다.
