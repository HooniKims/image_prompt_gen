Chrome Image Prompt Assistant

ChatGPT와 Gemini에서 이미지 생성용 프롬프트를 쉽게 만들고 현재 입력창에 넣어주는 Chrome 확장프로그램입니다.

설치 순서

  1. ZIP 파일을 압축 해제합니다.
  2. Chrome 주소창에 chrome://extensions를 입력합니다.
  3. 오른쪽 위 개발자 모드를 켭니다.
  4. 압축해제된 확장 프로그램 로드를 누릅니다.
  5. 압축 해제한 chrome-image-prompt-assistant 폴더를 선택합니다.

사용 방법

  1. ChatGPT 또는 Gemini 웹페이지를 엽니다.
  2. 오른쪽 패널에서 작업 유형, 주제, 스타일, 비율을 고릅니다.
  3. 프롬프트 만들기를 누릅니다.
  4. 자동 입력을 눌러 현재 입력창에 삽입합니다.
  5. 내용을 확인한 뒤 직접 전송합니다.

저장 원칙

작업 카드와 프롬프트는 브라우저 로컬 저장소에만 저장합니다. 생성 이미지는 확장프로그램이 자동 저장하지 않습니다.
